plexer.js


plexer.js expects a document structure similar to this one:


In the head:

<script src="http://code.jquery.com/jquery.min.js"></script>
<script src="src/plexer.js"></script>
<link rel="stylesheet" href="src/plexer.css">

In the body:

<div class="plexer">
	<ul id="switchboard">
		<li data-content-name="1">1</li>
		<li data-content-name="2">2</li>
		<li data-content-name="3">3</li>
	</ul>
	<div id="content">
		<div data-name="1">
			<h1>This is the first content div.</h1>
			<p>It has the <em>data-name</em> attribute set to <em>1</em>
		</div>
		<div data-name="2">
			<h1>This is the second content div.</h1>
			<p>It has the <em>data-name</em> attribute set to <em>2</em>
		</div>
		<div data-name="3">
			<h1>This is the third content div.</h1>
			<p>It has the <em>data-name</em> attribute set to <em>3</em>
		</div>
	</div>
</div>

Notice that the list items inside #switchboard all have a data-content-name attribute set, and that there's a div inside #content with a matching data-name attribute set. These values can be anything as long as they match. I picked 1, 2, and 3, but that's just how I was feeling at the time.
